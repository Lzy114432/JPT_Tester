数据流
硬件值 → [模拟] → simValue → [NO/NC映射] → logicalValue
                ↓                        ↓
             ctx.PreMap.属性           ctx.R.属性
实现要点
  1. 新增 Snapshot 存储 simValue
    - 在 Snapshot 中增加 _preMapInputs[] 数组
    - Tick 时同时保存 simValue（映射前）和 logicalValue（映射后）
  2. 新增 PreMap Layout 实例
    - 类似 _readLayoutFront，增加 _preMapLayoutFront / _preMapLayoutBack
    - Tick 时用 simValue 同步到 PreMap Layout
  3. 新增 PreMapAccessor
    - 类似 EdgeAccessor，暴露 ctx.PreMap 属性
    - 返回 _preMapLayoutFront
  使用效果
  // 逻辑值（经过模拟+映射）
  bool logical = ctx.R.急停按钮;
  // 原始值（经过模拟，绕过映射）
  bool preMap = ctx.PreMap.急停按钮;
